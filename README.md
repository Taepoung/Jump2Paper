# Paper2Web 📄✨

Paper2Web은 길고 정적인 학술 논문(Paper)을 **직관적이고 상호작용 가능한 단일 웹 페이지(Web)** 형태로 자동 변환시켜 주는 AI 에이전트 기반의 문서 파이프라인(Skill)입니다. 

단순히 텍스트를 복사해 붙여넣는 것을 넘어 수식 슬라이더, 데이터 플로우, 계층형 드릴다운 등의 동적 시각화 컴포넌트를 통해 독자가 논문의 핵심 아키텍처와 흐름을 직관적으로 "체감"할 수 있도록 돕습니다.

---

## 🚀 Core Features (주요 특징)

- **상호작용형 논문 탐색**: 스텝 플레이어(알고리즘 시연), 히트맵, 비교 슬라이더 등 논문의 개념을 직관적으로 설명하는 13가지 인터랙티브 컴포넌트 생태계를 갖추고 있습니다.
- **AI 친화적 파이프라인 `SKILL.md`**: 대형 언어 모델(LLM)이 논문을 완벽하게 소화하고 조립할 수 있도록 최적화된 **10단계**의 구조적인 작업 지침을 제공합니다.
- **무결점 단일 파일 빌드 (Single HTML)**: CSS, JavaScript 및 모든 컴포넌트의 로직이 내장된 가벼운 단일 HTML 형태로 배포됩니다.
- **최적화된 미디어 주입 (Context-Safe)**: 대용량 이미지(Base64)를 Python 스크립트로 직접 파일에 주입하여 AI 대화 내역(Context) 낭비를 막고 작업 속도를 극대화합니다.

## 📂 Directory Structure (구조)

```text
paper2web/
├── SKILL.md                          # [핵심] AI에게 지시하는 10단계 논문 변환 파이프라인 선언서
├── README.md                         # 현재 문서
├── assets/
│   └── components.html               # Paper2Web에서 사용되는 모든 HTML 템플릿/컴포넌트 원본 소스
└── references/
    └── design_system.md              # 색상, 토큰, 타이포그래피 및 용도별 컴포넌트 매핑(줄 번호) 가이드
```

### 각 참조 파일의 역할
*   **`SKILL.md`**: 작업자(AI)가 논문을 건네받아 완제품 페이지로 구워내기까지 거쳐야 하는 메인 프롬프트이자 규칙서입니다. 10단계의 정밀 공정으로 설계되었습니다.
*   **`design_system.md`**: "어떤 정보일 때 어떤 컴포넌트를 사용해야 하는가?"에 대한 판단 기준과, `assets/components.html` 내 정확한 행(Line) 번호가 명시되어 있습니다.
*   **`assets/components.html`**: 복사-붙여넣기를 위한 조각들의 집합소입니다. 전체 HTML 골격(Page Shell)부터 단위별 상호작용 컴포넌트까지 모두 정의되어 있습니다.

## ⚙️ How It Works (파이프라인 흐름)

AI 에이전트에게 변환이 시작되면 다음 순서에 따라 조립이 이루어집니다.

1.  **HTML 초기화**: `assets/components.html`에서 PAGE SHELL 코드를 추출하여 초기 뼈대 생성
2.  **디자인 시스템 로드**: `design_system.md`를 열람하여 컴포넌트 맵 및 행 번호 확인
3.  **논문 파싱 및 구조 파악**: 논문을 구조적으로 인지하며 본문과 Reference 분리
4.  **내용 작성 및 조립 (Assemble)**: 논문 문맥에 맞는 컴포넌트를 추출하여 채워 넣기
5.  **부록(Appendix) 보강**: 부록 데이터를 파싱해 본문의 원래 맥락 위치에 정밀하게 병합
6.  **전체 내용 및 UX 점검 (QA)**: 흐름 완결성, 스타일 충돌 여부 및 모바일 반응형 점검
7.  **최종 미디어 주입 (Python)**: 모든 검증 완료 후 Python 스크립트로 대용량 Figure 최종 주입
8.  **완결성 검증 및 빌드**: 링크 작동 여부, 레이아웃 충돌 등 치명적 UX 오류 최종 확인
9.  **배포 준비**: 논문 제목 슬러그로 파일명 변경 및 최종 결과물 전달

## 🏁 Getting Started (시작하기)

이 프로젝트는 Cursor, Claude, Antigravity 등 코딩이 가능한 **AI 에이전트(LLM)**를 대상으로 합니다.

AI 에이전트의 채팅 창이나 명령줄에서 다음과 같이 프롬프트를 입력하세요.

> **"@[논문 파일]을 읽고, @SKILL.md 에 기재된 파이프라인에 따라 인터랙티브 HTML 논문 페이지를 만들어줘."**
